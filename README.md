# Mr-Kelpy-AI-Robot
A driving robot with built in computer vision and an LLM using Raspberry Pi 5 and Arduino Uno Board

Components needed:

- 13 3D printed parts
- Various M4 and M5 screws
- Raspberry Pi 5
- Raspberry Pi Camera Module
- Arduino Uno Board
- Ultrasonic sensor
- 2X Servo Motors
- L298N Motor Driver
- 2X DC Motors
- 2X wheels
- 2X 360 rolling wheels
- Custom PCB for buck conversion and pin management
- 0.96 inch OLED Display 


Steps to project:

- Basic driving
- claw test
- wall detection and turn
- Eye reaction to touch
- object detection with pi camera and object grabbing
- Face detection 
- Voice recognition and= direct response
- Using ChatGPT APIs to import LLM
- Face Recognizer and greeting
- Magnetic charging
- Local AI Model
- Other features: display temperature, play music, etc.
